== Description ==

A simple plugin for adding an Basic Alert Box block to your site. This block gives you the capability to change the color, icon, and alignment. Comes with two different styles.

Interested in developing this block on your own? This block was developed as part of the [Complete WordPress Developer Course](https://www.udemy.com/course/wordpress-development-create-wordpress-themes-and-plugins/?referralCode=C65B44C0E4C6B06A7A92). Check it out if you would like to learn how to develop your own custom blocks. 

== Installation ==

= From Gutenberg Editor: =

1. Go to the WordPress Block/Gutenberg Editor
2. Search For **Basic Alert Box**
3. Click on the **Basic Alert Box** to add the block

= Download & Upload: =

1. Download the **Basic Alert Box** plugin (.zip file)
2. In your admin area, go to the Plugins menu and click on **Add New**
3. Click on **Upload Plugin** and choose the **alert-box.zip** file and click on **Install Now**
4. Activate the plugin and Enjoy!

= Manually: =

1. Download and upload the **Basic Alert Box** plugin to the **/wp-content/plugins/** directory
2. Activate the plugin through the **Plugins** menu in WordPress
